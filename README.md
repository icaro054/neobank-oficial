### Instale o Git Bash em sua máquina para rodar alguns comandos: 

**git clone "cole aqui a url do repositório"**
*ele vai clonar o repositório em sua máquina*

**code .**
*Vai abrir o VS Code direto no repositório*

*Dentro do VS Code, abra o terminal e alterne para o GitBah para inserir os seguintes comandos:*

**git config --global user.email "digite o seu email que foi vinculado ao GitHub"**

**git config --global user.name "digite o seu nome de usuário do GitHub"**
